PBPK Ontology: Description back to ToC

Physiologically Based Pharmacokinetic Models (PBPK) are widely used in both pharmaceutical and environmental domain by experts to assess the kinetics of xenobiotics. Thousands of papers have been published over years but the major drawback being utilization of inconsistent vocabulary which reduce the potential for automation. Machine-readable semantic technology like ontology can be used to improve the reporting of PBPK models. Currently there are multiple ontologies related to life science domain on portals like Bioportal, still there is no dedicated ontology for describing PBPK. The objective of this work is to develop a dedicated ontology for PBPK named PBPKO which can be used in future by wider kinetic community. PBPKO has covered possible mechanisms, existing kinetic terms in multiple databases and commonly utilized terminology by experts. This is available ontology with first version (V1.0) being published following FAIR guidelines using ROBOT tool and published at OBO foundry (https://github.com/InSilicoVida-Research-Lab/pbpko). It will be further enriched by including multiple species and much more kinetic terms with the help of experts. PBPKO annotates both physiological and biochemical parameters related to PBPK modeling. This ontology can be further used for text mining of PBPK parameters and model output.

Integration of PBPKO ontology with BFO

The PBPKO ontology is built upon the foundation of Basic Formal Ontology (BFO), which provides a robust philosophical framework for organizing biomedical entities. The integration follows these key relationships:

1. Continuants (BFO):
• Material entities: Organs, tissues, blood components
• Quality: Physiological parameters, biochemical properties
• Function: Absorption, distribution, metabolism processes

2. Occurrents (BFO):
• Processes: ADME mechanisms, enzymatic reactions
• Process boundaries: Temporal phases of drug distribution

This hierarchical structure ensures that PBPK concepts are organized in a standardized way, facilitating interoperability with other biomedical ontologies and enabling precise representation of pharmacokinetic knowledge.
